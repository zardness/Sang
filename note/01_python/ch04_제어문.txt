

4장. 제어문(fi, for while)
  1절. 조건문(if)
	if 조건1:
   		조건1이 참일 경우 수행할 명령어들(if문 단독 가능, 들여쓰기 중요)
	elif 조건2:
    		조건1이 거짓이고, 조건2가 참일 경우 수행할 명령어들(elif는 필요한 만큼 사용가능) 
	else :
    		모든 조건이 만족하지 않을때 수행할 명령어들(else는 단독사용불가)

  2절. 반복문
	
	for 임시변수 in 나열가능한자료:
    		반목할 문장들
	else: 
    		나열 가능한 자료들  다 소요하여 for블록을 다 수행한 후, 최종 한번 실행할 문장들
    		for문안의 break를 수행할 경우, else블록을 수행하지 않음(else는 선택)

       while 조건 : 
			반복할 문장들
	   else:
		반복을끝내고 한번 실행
	cf.break vs continue
