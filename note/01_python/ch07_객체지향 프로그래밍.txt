7 장 객체지향 프로그래밍

	-class : 속성(변수)과 메소드(함수)를 갖는 객체를 만들기 위한 설계도
		모든 클래스는 Object를 상속받는다.(__str__(self), ...)
	
	-class내에서는 
		생성자(생성자가 없으면 디폴트 생성자 자동생성) 및 소멸자
		인스턴스변수(맴버변수) VS 클래스 변수 VS static변수
		인스턴스메소드(일반함수) VS 클래스 메소드 VS static메소드
	
	-class 클래스명(상속받을 클래스명):
		클래스 변수 
		__static
		def__init__(self,[매개변수,...]}:
			self.인스턴스변수 = 매개변수 초기화
		def__del__(self):
			객체 소멸시 수행할 명령어
		def 인스턴스매소드명(self,[매개변수]):
			명령어
		@classmethod
		def클래스메소드명(cls,[매개변수,...]):
			cls.클래스변수와 관련된 로직(클래스명.클래스메소드명()호출)
		@staticmethod
		def static메소드명([매개변수,..]):
			코드
	
	-객체변수(인스턴스) = 클래스명(초기화될 맴버변수 값들)
		ex. 객체변수.__dict__객체를 딕셔너리 형태로 보여줌
	
	-상속 : 부모클래스로부터 자식 클래스에 모든 속성을 받는 것
		super():'부모클래스의' 를 의미
