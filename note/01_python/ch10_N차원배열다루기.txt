Numpy 패키지의 함수와 속성
	
	-배열 속성
		dtype : 요소들의 타입(타입 변경시 astype
		itemsize : 요소의 byte 사이즈
		ndim : 차원수(축수)
		size : 요소갯수
		shape : 배열의 구조(tup)
	-배열생성 함수
		np.array(obj,dtype,copy = T/F)
		np.array(obj,dtype,copy = T/F). reshape(행수,열수)
		np.zeros(tup,dtype)
		np.ones(tup,dtype)
		np.pull(tup,dtype)
		np.empty(tup,dtype)
