3장 데이터 구조 : 여러 데이터를 한 변수에 담는게 목적
1절.  리스트 : [타입에 상관없이 0개 이상의 데이터들] 
        1.1 리스트생성 : [ ],list(나열가능한 자료),(range(from,to,step))
        1.2 기본정보 조회
             -리스트 변수(lst)[인덱스]  = lst(idx]
             -min(lst), max(lst), sum(lst), len(lst),
             - sorted(lst,reverse=True) : 정열된 lst를 반환
             - lst.sort(reverse = True) : lst가 정열된 값으로 변경
        
        1.3. 요소 추가
             - append(value), extend([values]), insert(idx,value)

        1.4. 인덱싱과 인덱싱 찾기
              - ist[idx]
              - index (찾을 데이터) : 없는 데이터를 찾을경우 error
              - 찾을데이터 in lst
              -lst,count(찾을 데이터) : 찾을 데이터가 몇개인지 숫자 반환

        1.5 원하는 요소 추출
              -슬라이싱
              -리스트컴프리헨션(원하는 데이터 추출)

        1.6 요소 수정 : 인덱싱이나 슬라이싱을 이용 

        1.7 요소 삭제 : pop( ), pop(idx), del lst[idx], remove(데이터)

        1.8 복제 (깊은 복사): 복제할 변수 = 원본데이터.copy( )
