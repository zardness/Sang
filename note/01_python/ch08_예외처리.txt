8장 예외처리

	예외처리는 보험이다.
	실행 중 발생하는 예기치 않은 사건을 대비하는 구현

-문법
 try:
    예외가 발생할수도 있는 명령어들
except [예외타입1 [as e]]: # 여러except절이 있을 경우 상위 예외 타입은 아래로
    해당 예외타입시 수행할 명령어
except [예외타입2 [as e]]: # 여러except절이 있을 경우 상위 예외 타입은 아래로
    해당 예외타입시 수행할 명령어
[else:    try블록을 다 수행한 후 실행할 명령어 ]
finally:  try블록이나 except블록을 수행한 후 실행

-raise 예외타입() : 강제로 예외발생
	ex.중첩루프탈출,추상클래스(상속받은 클래스에서 함수를 재정의 하도록)
